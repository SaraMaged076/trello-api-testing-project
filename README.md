
# Trello API Automation Testing Project

This project demonstrates an end-to-end API testing project based on Trello APIs, focusing on simulating real-world workflows and ensuring reliable automation practices.

## Features:
- Automated CRUD operations (Create → Read → Update → Delete) for (Boards, Lists, Cards, Checklists)
- Request chaining using dynamic variables
- Basic performance checks (response time validation)
- Automated test scripts for validating responses and data integrity
- Negative testing for different scenarios (like 404)
- Designed a test execution schedule to control request dependencies

## Tools:
- Postman
- Trello REST API

## How to Run:
1. Import the collection JSON into Postman
2. Add your API key and token
3. Run using Collection Runner
